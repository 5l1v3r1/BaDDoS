# BaDDoS
BaDDoS is an **advanced** program to perform DoS attack based on ping.

# Usage
Launch:
> release/BaDDoS.exe

# Features
 1. **Multithreading**
 2. Presets
 3. Tools: "**IP Scanner**", "**Ping**", "**WhoIs**", "**Header**", "**My IP**"
 4. Many options
